Requisitos y Pasos para Acceder

📋 Para Estudiantes:

  • Email institucional UNSJ (@unsj.edu.ar)
  • Cuenta GitHub activa
  • Verificación de estado estudiantil
  • Documento que acredite inscripción actual

🎓 Para Profesores:

  • Email institucional UNSJ (@unsj.edu.ar)
  • Cuenta GitHub activa
  • Verificación de empleo académico
  • Documentación que acredite cargo docente

🔗 Proceso de solicitud:

1. Visita: education.github.com
2. Aplica para "GitHub Student Developer Pack" o "GitHub Teacher Toolbox"
3. Completa verificación institucional
4. Activa GitHub Copilot en tu cuenta
